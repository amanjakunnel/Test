# HW3 Readme

## Libraries to be installed:

- PyTorch (pip3 install torch)
- torchvision (pip3 install torchvision)


## Virtual Environment Setup:
```bash
python3.11 -m venv .venv
source .venv/bin/activate
```

## Command to run code:
```bash
python3 extract_resnet18_params.py
```

## Output Storage Location:
Filename - `resnet18_params.csv` 
Location - Same directory where the script is run
